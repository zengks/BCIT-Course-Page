Welcome to COMP 1950

This practical course will provide more in-depth training regarding HTML and CSS along with a bit touch-up on Javascript and PHP. Successful students of this course will gain a deeper understanding of the latest technologies including HTML5 and CSS3 for developing web sites and applications with a focus on responsive design using Media Queries.
Both prospective and current students are welcome to join this course with the following requirements:
Prospective Students
COMP1950 has a pre-requisite of COMP1850. It means new students cannot register this course without completing COMP1850 unless the students can proof themselves already understand materials covered in COMP1850. More details can be found on BCIT website.
Current Students
Current students who already pass COMP1850 are eligible to register COMP1950 as long as seats are available.
